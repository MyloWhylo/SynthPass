SynthPass Web Flasher

Flash a CH572 over WebUSB. Chromium only.
WebUSB unavailable. Open this page in a Chromium based browser instead (Firefox / Safari don't expose WebUSB).
1. Pick a firmware.bin (PlatformIO writes one to .pio/build/genericCH572Q/firmware.bin).
2. Click Flash. 3. Put the board into ISP bootloader mode by holding down the "Boot" button while you plug it in.
4. Grant the browser access to the device when prompted.
